Crear diccionario¶

Las listas countries y capitals están nuevamente disponibles en el script. Es su trabajo convertir estos datos en un diccionario donde los nombres de los países son las claves y las mayúsculas son los valores correspondientes. Como repaso, aquí hay una receta para crear un diccionario:

python my_dict = {      "key1": "value1",      "key2": "value2",   }

En esta receta, tanto las claves como los valores son cadenas. Este también será el caso para este ejercicio.

Actividad

  • Imprima europe para ver si el resultado es el que esperaba.

Acceso al diccionario¶

Si las claves de un diccionario se eligen sabiamente, acceder a los valores de un diccionario es fácil e intuitivo. Por ejemplo, para obtener la capital de Francia europe, puede usar:

europe['france']

Aquí ‘france’ está la clave y ‘paris’ es el valor devuelto.

Actividad

  • Compruebe qué claves se encuentran europe llamando al método keys() sobre europe. Imprima el resultado.

  • Imprima el valor que pertenece a la clave ‘norway’.

Respuesta:
dict_keys([‘spain’, ‘france’, ‘germany’, ‘norway’])
oslo

Si sabe cómo acceder a un diccionario, también puede asignarle un nuevo valor. Para agregar un nuevo par clave-valor europe, puede usar algo como esto:

europe['iceland'] = 'reykjavik'

Diccionario de diccionarios¶

¿Recuerdas las listas? Podrían contener cualquier cosa, incluso otras listas. Bueno, para los diccionarios lo mismo vale. Los diccionarios pueden contener nuevamente diccionarios.

Como ejemplo, eche un vistazo al script donde europe está codificada otra versión del diccionario con el que ha estado trabajando todo el tiempo. Las claves siguen siendo los nombres de los países, pero los valores son diccionarios que contienen más información que solo la capital.

Es perfectamente posible encadenar corchetes para seleccionar elementos. Para buscar la población de España europe, por ejemplo, necesita:

europe['spain']['population']

Bucle sobre el diccionario¶

En Python 3, necesita el método items() para recorrer un diccionario:

world = { "afghanistan":30.55,
    "albania":2.77,
    "algeria":39.21 }

for key, value in world.items() :
    print(key + " -- " + str(value))
