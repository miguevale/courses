Listas y tuplas¶

90 min | Última modificación: Junio 07, 2020.

Listas¶

Las listas permiten guardar más de un elemento dentro de una variable en un orden concreto. Pueden contener un número ilimitado de variables de cualquier tipo:

[1]:

alumnos = ["Calos", "Sandra", "Lorenzo", "Laura"]
print(alumnos [0]) # muestra "Calos"
print(alumnos [1]) # muestra "Sandra"
print(alumnos [2]) # muestra "Lorenzo"
print(alumnos [-1]) # muestra "Laura"

Calos
Sandra
Lorenzo
Laura

Los métodos más utilizados con las listas son los siguientes:

list.append(x) – agrega el elemento x al final de la lista.

list.extend(L) – agrega la lista L al final de la lista.

list.insert(i,x) – inserta el elemento x en la posición i.

list.remove(x) – remueve el elemento x.

list.pop([j]) – remueve el elemento en la posición j de la lista.

list.clear() – borra la lista; elimina todos los elementos de la lista.

list.index(x) – devuelve el indice del elemento x

list.count(x) – Cuenta las veces que aparece el elemento x.

list.sort(key=None, reverse=False) – ordena los elementos de la lista.

list.reverse() – Inverte el orden de los elementos de la lista.

list.copy() – crea una copia de la lista.

alumnos = ["Calos", "Sandra", "Lorenzo", "Laura"] # Creación de la lista
print(alumnos)

['Calos', 'Sandra', 'Lorenzo', 'Laura']

['Sandra', 'Lorenzo', 'Laura']

Subconjunto de lista¶

Seleccionar valores individuales de una lista es solo una parte de la historia. También es posible dividir su lista, lo que significa seleccionar varios elementos de su lista. Use la siguiente sintaxis:

my_list[inicio:fin]

El índice inicio se incluirá, mientras que el índice fin no.

El siguiente ejemplo de código muestra un ejemplo. Se selecciona una lista con “b” y “c”, correspondiente a los índices 1 y 2, de una lista x:

[24]:

x = ["a", "b", "c", "d"]
x[1:3]

#Los elementos con índice 1 y 2 están incluidos, mientras que el elemento con índice 3 no lo está.

[24]:

['b', 'c']

También es posible no especificar estos índices. Si no especifica el índice inicio, Python seleccionará la sublista desde el primer elemento de la lista. Si no especifica el índice fin, la sublista irá hasta el último elemento de su lista.

[25]:

x = ["a", "b", "c", "d"]

print(x[:2])
print(x[2:])
print(x[:])

['a', 'b']
['c', 'd']
['a', 'b', 'c', 'd']

Subconjunto de lista de listas¶

Viste antes que una lista de Python puede contener prácticamente cualquier cosa; incluso otras listas! Para subconjuntar listas de listas, puede usar la misma técnica que antes: corchetes. Pruebe los comandos en el siguiente ejemplo de código en las celdas de jupyter:

x = [["a", "b", "c"],
     ["d", "e", "f"],
     ["g", "h", "i"]]
x[2][0]
x[2][:2]

[26]:

x = [["a", "b", "c"],
     ["d", "e", "f"],
     ["g", "h", "i"]]
print(x[2][0])
print(x[2][:2])
print(x[2]) # da como resultado una lista que puede volver a subconjuntar agregando corchetes adicionales.
print([x[2]])

g
['g', 'h']
['g', 'h', 'i']
[['g', 'h', 'i']]

Desempaquetado de listas¶

Una acción típica de las listas es “desempaquetar” (unpack) sus valores.

[27]:

a, *rest = [1, 2, 3]    # a = 1, rest = [2, 3]
print(rest)

[2, 3]

[28]:

a, *middle, c = [1, 2, 3, 4] # a = 1, middle = [2, 3], c = 4
print(middle)

[2, 3]

Recorrer una lista¶

La forma habitual de recorrer una lista es mediante la sentencia for, tal y como muestra el ejemplo a continuación:

[29]:

for elemento in ['Python','Excel','R']:
    print("Programo en ", elemento)

Programo en  Python
Programo en  Excel
Programo en  R

De igual manera se podría hacer mediante la sentencia while:

[30]:

lista = ['Python','Excel','R']
i = 0
sizeofList = len(lista)
while i < sizeofList :
    print(lista[i])
    i += 1

Python
Excel
R

Tuplas¶

Las tuplas son listas inmutables. Es decir, una vez declaradas, no se pueden realizar modificaciones sobre ellas (añadir/eliminar elementos o hacer modificaciones sobre ellos). Para definir una tupla se escriben los elementos separados por comas:

tupla = 1,2,3

El acceso a sus elementos se hace de igual que con listas:

[31]:

tupla = "a", "b", "c","d","e","f"
print(tupla) # una tupla se diferencia de una lista porque se imprime entre paréntesis.
print(tupla[1])
print(tupla[2:4])

('a', 'b', 'c', 'd', 'e', 'f')
b
('c', 'd')

Una acción típica de las tuplas es “desempaquetar” (unpack) sus valores:

[32]:

tupla = (1,2,3)
a, b, c = tupla # asignación de los elementos de una tupla.
print(a)
print(b)
print(c)

1
2
3

Tupla de tuplas¶

Al igual que la lista de listas, también se puede subconjuntar en tupla de tuplas.

[33]:

# Las tuplas pueden anidarse.
u = tupla, (1, 2, 3, 4, 5)
u

[33]:

((1, 2, 3), (1, 2, 3, 4, 5))

La función len() devuelve el número de elementos de una lista o tupla:

[34]:

alumnos = ["Calos", "Sandra", "Lorenzo", "Laura"]
tupla = (1,2,3)
print(len(alumnos))
print(len(tupla))

4
3

Conjuntos¶

Un conjunto es una colección no ordenada y sin elementos repetidos. Los conjuntos también soportan operaciones matemáticas como la unión, intersección, diferencia, y diferencia simétrica.

[35]:

canasta = ['manzana', 'naranja', 'manzana', 'pera', 'naranja', 'banana']
fruta = set(canasta)
print(fruta)

{'banana', 'naranja', 'pera', 'manzana'}
