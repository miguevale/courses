Estructuras de control¶

90 min | Última modificación: Junio 07, 2020.

Condicionales¶

Las estructuras de control se utilizan para ejecutar bloques de código en función de condiciones.

Estructura if - else¶

Se evalúa la condición especificada en la sentencia if y en caso de cumplirse se ejecutará el bloque de código indentado. En caso de que el resultado de la condición sea False, no el bloque especificado no se ejecutará.

La identación usando cuatro espacios en blanco es el mecanismo definido en Python para delimitar el cuerpo asociado a las estructuras de control y a las funciones de usuario.

[1]:

numero = 5
if numero > 1:
    print("Es mayor que uno")

Es mayor que uno

Es posible especificar un bloque de código que se ejecute en caso de que la condición no se cumpla:

[2]:

numero = 2
if numero > 10:
    print("Es mayor que diez")
else:
    print("Es menor o igual que diez")

Es menor o igual que diez

También podemos comprobar más condiciones mediante la expresión elif. En este caso, se seguirán comprobando todas las condiciones elif hasta que una de ellas se cumpla. En caso contrario, se ejecutará el bloque de código dentro de else (si lo hubiera).

El uso del caracter : es obligatorio.

[3]:

# este código es mucho más dificil de leer.
numero = 5
if numero < 3:
    print("Es menor que 3")
else:
    if numero < 6:
        print("El número está entre el 3 y el 5")
    else:
        print("Es mayor o igual a 6")

El número está entre el 3 y el 5

[4]:

# código mucho más legible
numero = 5
if numero < 3:
    print("Es menor que 3")
elif numero < 6:
    print("El número está entre el 3 y el 5")
else:
    print("Es mayor o igual a 6")

El número está entre el 3 y el 5

Bucles¶

Los bucles permiten ejecutar un bloque de código tantas veces como queramos.

Estructura while¶

El ciclo while es como una declaración if repetitiva. El código se ejecuta una y otra vez, siempre que la condición sea True. Echa otro vistazo a su receta.

while condition :
    expression

[5]:

n = 0
while n < 5:  # se ejecuta mientras se cumpla que n < 5
    print(n)
    n = n + 1

print('fin')

0
1
2
3
4
fin

Comandos break, continue y pass¶

  • El comando break causa la salida del cuerpo del ciclo while/for.

  • El comando continue causa que se ejecute una nueva iteración del ciclo while/for sin pasar por el resto del código que hace parte del cuerpo del ciclo while/for.

  • La sentencia pass no hace nada. Se puede usar cuando una sentencia es requerida por la sintáxis pero el programa no requiere ninguna acción.

[9]:

n = 0
while n < 5:
    n = n + 1
    if n == 3:
        break
        print(n)
    else:
        print(n)
print('fin')

fin

Bucle while/for con else¶

Los ciclos creados usando los comandos for y while pueden contener un comando else. En el caso de los ciclos for, el cuerpo del else se ejecuta cuando se termina el ciclo; para los ciclos while, el cuerpo del else se ejecuta cuando el condicional se hace False.

[12]:

n = 0
while n < 5:  # se ejecuta mientras se cumpla que n < 5
    print(n)
    n = n + 1
else:
    print('cuerpo else')

print('fin')

La función range()¶

La función range([start,] stop [, step]) devuelve una secuencia de números. Es por ello que se utiliza de forma frecuente para iterar:

[14]:

for i in range(3):
    print(i)

0
1
2

También podemos indicar el inicio, fin y step:

[15]:

print("Números del 5 al 10")
for i in range(5, 10):
    print(i, end=', ')

Números del 5 al 10
5, 6, 7, 8, 9,

[16]:

print("Números impares del 1 al 10")
for i in range(1, 10, 2):
    print(i, end=', ')

Números impares del 1 al 10
1, 3, 5, 7, 9,

Para iterar por una lista utilizando el índice, basta con combinarlo con la función len():

[17]:

alumnos = ["Ane", "Mikel", "Unai", "Lorea"]
print(len(alumnos))
for i in range(len(alumnos)):
    print(alumnos[i])

4
